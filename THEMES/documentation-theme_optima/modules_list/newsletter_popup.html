Configure Pos newsletter popup

This allows you to to config selected elements as general configuration for Pos newsletter popup module.

NOTE: This module requires Newsletter subscription installed. Please sure that you installed Newsletter subscription module.

  1. Login your Prestashop admin

  2. Go to POSTHEMES > Modules > Popup

  3. We got:

    We got:

    • Preset layout: Select preset layout for newsletter popup.
    • Popup width: Popup window width. Below this width module will be hidden.
    • Popup height: Popup window height. Below this height module will be hidden.
    • Title content: Title of content of popup.
    • Custom content of popup module: Add custom content to the newsletter popup
    • Order by: Change the order of your products
    • Popup background color: Set the background color
    • Popup image: Click "Select image' to upload background image.
    • Text color: Select color of text in popup.
    • Popup delay: Delay show of popup for x miliseconds
    • Cookie time: Time in days of storing cookie. After that time windows will be showed again

    => Configure parameters as you want

  4. Click Save button when you finish

Translate module

  • Login your Prestashop admin

  • Go to POSTHEMES > Modules > Popup

  • Click to choose language you want to translate.
  • At Title content, Custom content of popup module: translate text.
  • Click Save button when you finish
