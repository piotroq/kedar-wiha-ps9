Configure Pos Themeoptions module

General Configuration

  1. Login your Prestashop admin

  2. Go to POSTHEMES > Modules > Theme settings

  3. In GENERAL tab

    We got:

    * In Layout part:

    • Layout: Select wide or boxed for layout
    • Container max width : Set maxium width of page. You must provide px or percent suffix (example 1240px or 100%)
    • Sidebar width : Choose width of sidebar with 3 options: normal, narrow, wide.

    * In Body font & color part:

    • Main color: Select main color for this theme
    • Google font URL: Add google font URL(Example: https://fonts.googleapis.com/css?family=Open+Sans:400,700 . Add 400 and 700 font weigh if exist. If you need adds latin-ext or cyrilic too. Go to Google font to get font URL )
    • Google font name: Add Google font name (Example: 'Montserrat', sans-serif )
    • Body font size: Font size of body
    • Body font color: Select color of body

    * In Body background part:

    • Body background color: Select the background color of body
    • Body background image : Select the background image of body
    • Body background repeat: Set how background images are repeated. A background image can be repeated along the horizontal and vertical axes, or not repeated at all
    • Body background attachment : Set whether a background image scrolls with the rest of the page, or is fixed
    • Body background size: Select the size of the background images: auto, cover

    * In Title block part:

    • Title block- Google font URL: Add google font URL(Example: https://fonts.googleapis.com/css?family=Open+Sans:400,700 . Add 400 and 700 font weigh if exist. If you need adds latin-ext or cyrilic too. Go to Google font to get font URL )
    • Title block- Google font name: Add Google font name (Example: 'Montserrat', sans-serif )
    • Title block font size: Font size of the title
    • Title block color: Select color of the title
    • Title block font-weight: Select font-weight of the title
    • Title block transform: Select text-transform for title (None/Capitalize/UPPERCASE)
    • Title block fontsize in column: Font size of the module's title, that modules can be placed within any column position.

    => Configure parameters as you want

  4. Click Save button when you finish

Page Title

This allows you to choose background image, text color, title size of other pages in theme.

  2. Go to POSTHEMES > Modules > Theme settings

  3. In PAGE TITLE tab

    We got:

Product Grid Settings

  1. Login your Prestashop admin

  2. Go to POSTHEMES > Modules > Theme settings

  3. In PRODUCT GRID SETTINGS tab

    We got:

    • Product grid display: Select style to display buttons on product
    • Use no border: Select Yes if you don't want to show border arround product box.
    • Use no padding: Select Yes if you don't want to have padding between products
    • Disable quickview: Select Yes if you don't want to show quickview
    • Product name color: Color of product name
    • Product name color hover: Color of product name when hover
    • Product name font weight: Choose font-weight of product name
    • Product name font size: Set font size of product name
    • Product name length: Select 2 options for length of product name: 1 line or 2 lines
    • Product name transform: Select text-transform for the product name (None/Capitalize/UPPERCASE)
    • Price color: Select color of the price
    • Price font size: Font size of the price
    • Show brand: Select Yes if you want to have show brand for each product
    • Show rating: Select Yes if you want to show rating for each product

    => Configure parameters as you want

  4. Click Save button when you finish

Product Page Settings

  1. Login your Prestashop admin

  2. Go to POSTHEMES > Modules > Theme settings

  3. In PRODUCT PAGE SETTINGS tab

    We got:

    In Product page layout :

    • Product layout: Select layout for product detail page
    • Product image: Product thumbnails can be displayed horizontal slider(thumbnails left/right) or vertical slider(thumbnails bottom), grid, one column.
    • Product infomation tab display: Product infomation tab can be displayed horizontal/vertical/accordion
    • Product infomation position: Select position of product information

      In Configurations :

    • Product name color: Select color of the product name
    • Product name font size: Font size of the product name
    • Product name transform: Select text-transform for the product name (None/Capitalize/UPPERCASE)
    • Price color: Select color of the price
    • Price font size: Font size of the price
    • Hide buy now button: Choose Yes to hide Buy Now button in the product page.

    => Configure parameters as you want

  4. Click Save button when you finish
