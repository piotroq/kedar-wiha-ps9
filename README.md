# kedar-wiha-ps9
 System PrestaShop 9.0.3 w oparciu o motyw premium "Optima Wersja 3.3.0" katalog "theme_optima". Ten projekt ma za zadanie rozbudowywanie tego motywu "Optima Wersja 3.3.0" za pomocą tworzenia nowych modułów i generowaniu custom CSS/JS do implementacji na produkcji do motywu. Sklep finalnie ma być tylko w języku polskim.
